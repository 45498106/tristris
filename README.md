### Change log:

#### 0.1.0-alpha (July 07, 2017):

- Basic tetris-like gameplay
- Scoring system ($10^{rows\ cleared}$ points per move, and `score`/`highscore` display)
- Background music, basic sfx
- Controls:
    - Move: `←`, `↑`, `→`, `↓`
    - Hold: `Shift`
    - Pause: `P`
    - Mute: `M`
    - Skip Track: `S`
